Exercício 13

O que é estilização?

A estilização de uma página web é realizada através do CSS, definindo design, layout e a exibição dos seus elementos, a depender do tamanho de tela disponível.

Qual é a semântica do CSS?

O estilo de uma página pode estar no próprio arquivo HTML, de forma interna, ou como um arquivo em anexo com extensão .css, de forma externa.

Tipos de seletores CSS:

  1. Por 'id': .id_do_elemento {} - este id só pode ser utilizado por um único elemento;
  2. Por 'class': #classe_do_elemento {} - esta classe pode ser usada por vários elementos;
  3. Por elemento: p {} - altera todas as ocorrências do elemento mencionado.
